## User Stories

Une User Story est une description simple d'une fonctionnalité. Elle est rédigée sous la form d'une phrase simple sur le modèle suivant :

```text
En tant que (type utilisateur), je souhaite (fonctionnalité) afin de (bénéfice attendu)
```
